Frequently Asked Questions

What is this solution for?

Write a function, undirectedPath, that takes in an array of edges for an undirected graph and two nodes (nodeA, nodeB). The function should return a boolean indicating whether or not there exists a path between nodeA and nodeB.

What I did beyond what was asked?

This webpage shows beyond what the problem asked for. The standard undirected path problem checks if there's a path between two nodes. But on this GraphQuest, I infuse magic into your graphs. Watch as nodes come to life, edges illuminate, and the entire graph dances to the rhythm of connectivity.
